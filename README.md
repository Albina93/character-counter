# Character Counter

A real-time character counter application built with React and Typescript.

## Features

- Real-time character count
- Real-time word count
- Estimated reading time
- Minimum and Maximum word count goals

## Component overview

### CharacterCounter

The main parent component that manages state and connects all components together.

### TextInput

A textarea component that captures user input and sends it up to the parent via callback function.

### StatsDiplay

A display component that receives statistics as props and shows them to the user.
